## Condicional IF / ELSE

### IF
```py
    x = int(input("Asigna el valor de X: "))  # Permite que el usaurio introduzca un valor en la terminal

    if x > 9:
        print("X es mayor que 9")
        
```

### ELSE
```py
# Asignación de variables
x = (int(input("Introduce el valor de X: ")))

if x > 9:
    print("X es mayor que 9")
else:
    print("X es menor que 9")

print("Fin del programa.")
```

### ELIF

```py
# Asignación de variables
x = (int(input("Introduce el valor de X: ")))

if x < 9:
    print("X es menor que 9")
elif x < 15:
    print("X es menor que 15")
else:
    print("X es mayor que 15")
print("Fin del programa.")
```
